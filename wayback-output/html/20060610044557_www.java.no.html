Google lanserer AJAX toolkit
Leif J, 19. mai 2006 10:44
Google har helt siden første versjon av sin eposttjeneste GMail vært langt fremme i bruk av Ajax-teknologi. I senere tid har tjenester som Google Maps og Google Notepad forsterket Googles posisjon i dette segmentet. I forbindelse med JavaOne har firmaet nå lansert et verktøy for koding av Ajax-baserte applikasjoner. Google Web Toolkit baserer seg på en noe uventet tilnærming. Man koder nemlig hele applikasjonen i java i sin egen favoritt-IDE. Når applikasjonen er ferdigutviklet kjøres javakoden gjennom en java-til-javascript-prosessor, som så genererer html og javascript som kan deployes på webserveren. Bruken av java medfører at man kan bruke et IDE man er vant med, i tillegg til at man har tilgang på vanlig infrastruktur som debugger, JUnit osv. Google Web Toolkit tilbyr også widgets for bruk i applikasjonen, så som menylinjer, modale dialoger, herarkiske trestrukturer etc.
