JSR 244 - Java 2 Platform, Enterprise Edition 1.5 (J2EE 1.5)
Leif J, 29. april 2004 09:52:33
Sun har registrert JSR 244 - Java 2 Platform, Enterprise Edition 1.5 (J2EE 1.5). Sun skriver at denne JSR skal benyttes i utviklingen av den neste versjonen av J2EE, som etter planen skal være klar andre halvdel 2005. Fokus for spesifikasjonen vil være å gjøre utviklingen lettere. JSR-175 ( Metadata ) vil være sentral i utviklingen av en ny type deklarativ programmering som signifikant vil forenkle utviklingsprosessen for J2EE-utviklere. J2EE 1.5 vil basere seg på J2SE 1.5 (Tiger) og bl.a inneholde en integrasjon av JSF og JSP, EJB 3.0, JAXB 2.0 og JAX-RPC 2.0. Så får vi se om Sun treffer bedre med tidsestimatene sine denne gangen.
