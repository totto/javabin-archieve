Medlemsmøte i oktober

Vi er desverre nødt til å avlyse denne månedens medlemsmøte. Dette delvis på grunn av høstferien og delvis dårlig planlegging fra denne månedens møteleder (Markus). Vi kommer strekere tilbake 2. november med "J2EE i praksis".

Web-dugnad

Mandag 18.9 ble det gjort en dugnad her på www.java.no. Tusen takk til nye og gamle javaBin-helter. Send en mail til styret@java.no dersom du har spørsmål.

Medlemsmøte september

Torsdag 7.9 møtes vi på Ifi, 18:00 med følgende temaer: MikroJava, status og erfaringer. Øyvind Wergeland, Infostream. Stikkord: Egenskaper ved siste versjon(er), eksempler med JavaCard m.m. Profilering og minnehåndteringsverktøy for Java. Arild Magnus Palencia, Compuware. Stikkord: Måling/detektering av flaskehalser, minneforbruk, etc.

Nytt og spennende på Java fronten

Torsdag 3. august, 18:00 braker det løs for høstsesongen i JavaBin og hva er mere naturlig enn å starte med en teknologi-oppsummering fra JavaOne? Mogul vil komme og fortelle oss og demonstrere nyhetene i EJB 2.0 Spesifikasjonen. E-Line Group vil fortelle om arbeidet med JDK 1.4 (Merlin). Til slutt vil det bli en debatt på Microsoft sine siste krumspring, nemlig C#.

javaBin på JavaOne-2000

I Juni er store deler av norske java-utviklere i San Fransisco på JavaOne. Send en mail til styret@java.no, så blir du informert om en liten uoffissiell norsk Java-event under JavaOne.

Medlemsmøte torsdag 4.mai

Torsdag 4.mai klokken 18:00 møtes vi igjen i lille auditorium, Institutt for Informatikk, Universitetet i Oslo. Først vil Erlend Ødegård (Anderson Consulting) fortelle om erfaringer med XML og Java. Deretter snakker Erlend Nossum (Gan Media) om JavaCC. Kveldens siste foredrag holdes av Øystein Myhre fra Telenor FoU som vil fortelle om bruk av Jini-teknologi for å lage distribuerte løsninger. For planleggingens skyld oppfordrer vi til forhåndspåmelding til styret@java.no

Java brukergrensesnitt og UML

Torsdag 6. April møtes vi på IFI klokken 18:00 for å høre på følgende spennende foredrag: Brukersnitt fra spesifikasjon til implementasjon. Foredragsholder: Jon Skandsen. Deretter stiller Mogul.com med Trond Andersen som vil vise hvordan man praktisk går mellom UML og Java i Rational Rose. Kvelden avsluttes med vår gamle kjenning, Trygve Reenskaug som vil prate om hvordan man kan se på Enterprise Java Beans som en UML collaboration.
