# Norwegian Java Ecosystem - Companies and People

## 1. ObjectWare (Totto's Employer)

- **Founded:** 1995 in Norway by Arne Mjos
- **1999:** Separate entity Itera founded; Arne Mjos went from Objectware CEO to Itera COO
- **2000:** Itera merged with Objectware; became Itera Consulting
- **2003-2009:** Totto worked as Sjefskonsulent (Chief Consultant) and fagansvarlig
  (Technical Lead) for the Java department
- **2008:** Objectware renamed to Itera Consulting
- **~2009:** Totto left during reorganization
- **Current:** Part of Itera ASA (Oslo Stock Exchange listed), 500+ employees
  across Norway, Sweden, Denmark, Ukraine, Slovakia

**IMPORTANT:** ObjectWare and ObjectNet are DIFFERENT companies despite similar names.

## 2. ObjectNet AS

- **Founded:** 1999 in Oslo, IT consultancy specializing in Java and Open Source
- Clients: NSB, Telenor, Norsk Hydro, Statsbygg (major public sector)
- Grew to 80 consultants in Oslo
- **~2007:** Acquired by Swedish Know IT AB for **NOK 100 million** (80%)
- Became Knowit Objectnet AS, later Knowit Solutions Data & Tech AS
- Key people: Tobias Torrissen (CTO 2008-2015), Stein Grimstad (consultant),
  Fredrik Vraalsen, Alf Kristian Stoyle
- javaBin connection: Kjetil Jorgensen-Dahl represented ObjectNet on the board

Note: objectnet.dev is an UNRELATED computer vision dataset project.

## 3. The Cantara Connection

**Cantara** (Arabic for "bridge") is a Norwegian open source community
established in **2014** by Thor Henning "Totto" Hetland.

**Origin story - directly tied to Objectware:**
- Objectware maintained an internal Community Wiki with extensive knowledge
- When Totto left during the Itera reorganization (~2009), content was at risk
- Totto arranged to have the entire knowledge base open sourced
- In 2014, Cantara was formally established to steward this content
- Wiki expanded to **900+ articles** on software architecture and engineering
- GitHub org (github.com/Cantara) now has **143+ repositories**

**Key Cantara projects:**
- Whydah - IAM/SSO platform (flagship)
- Xorcery - Microservices library (maintained by Rickard Oberg of eXOReaction)
- Visuale - Real-time dashboard visualization

**The thread:** Objectware knowledge base → open sourced by Totto → Cantara
community → eXOReaction builds on Cantara infrastructure. This explains why
the lib-pcb codebase uses package `no.cantara.pcb.lib`.

## 7. WM-Data → Logica → CGI

- Swedish IT consultancy with ~10% of turnover from Norway
- javaBin: Oyvind Lokling and Harald Kuhr both represented WM-Data (2005-2006)
- **2006:** Acquired by British LogicaCMG
- **2008:** Fully incorporated into Logica

## 13. Key Insight

**javaBin's 2005 board perfectly maps the consulting companies:**
- ObjectWare → Totto (president)
- ObjectNet → Kjetil Jorgensen-Dahl
- Mesan → Odd Erik Zapffe
- WM-Data → Oyvind Lokling, Harald Kuhr

These consulting companies literally sent their people to run javaBin. The
Norwegian Java community was built by consultants who competed commercially
but collaborated on community infrastructure. This is the Norwegian model -
"dugnad" (voluntary community work) applied to technology.
